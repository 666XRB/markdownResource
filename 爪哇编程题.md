## 思路？

> 在编写有关类的编程题时，通常涉及以下几个方面：
>
> 1. **定义类和成员变量：** 首先，明确定义**类的目的和属性**。确定**类的成员变量（字段**），这些变量将存储对象的状态。
> 2. **实现构造方法：** 编写构造方法，用于初始化对象的属性。构造方法通常用于在对象创建时设置初始状态。
> 3. **编写成员方法：** 实现类的功能性方法，这些方法用于执行特定的操作。这可能包括计算、修改对象状态或执行其他与类相关的任务。
> 4. **提供访问器和修改器方法：** **提供 getter 和 setter 方法**，以允许**外部代码访问和修改类的私有成员**变量。。
> 5. **实现类的测试：** 编写测试类，测试类的各种方法确保它们按预期工作。测试通常包括验证构造方法、成员方法以及 getter 和 setter 方法的正确性。
>
> ---
>
> 小tips？
>
> **成员变量使用 private 修饰：** 将类的成员变量声明为私有的，然后**通过公共的 getter 和 setter 方法提供访问**。

## (1)定义一个**人类**，包括属性：姓名、性别、年龄、国籍；包括方法：吃饭、睡觉、工作。

```java
package hello.main;

public class Person {
    //类的一些属性和变量
    //——+成员变量使用 private 修饰
    //依次为姓名 年龄 国籍 性别
    private String name;
    private int age;
    private String country;
    private String gender;

    //这是类的有参构造方法。。。
    public Person(String name, int age, String country, String gender) {
        this.name = name;
        this.age = age;
        this.country = country;
        this.gender = gender;
    }

    // 成员变量对应的getter方法，因为成员变量是私有的，外部想要访问就要用到这些方法。。
    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }

    public String getCountry() {
        return country;
    }

    public String getGender() {
        return gender;
    }

    // 成员变量对应的setter方法，虽然我们有对应的有参构造函数，但是要修改单独的属性的适合，我们用对应的setter方法也是一件美事
    public void setName(String name) {
        this.name = name;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public void setCountry(String country) {
        this.country = country;
    }

    public void setGender(String gender) {
        this.gender = gender;
    }

    // 题上要我们写的一些方法。。。
    void eat() {
        System.out.println("吃饭");
    }

    void sleep() {
        System.out.println("睡觉");
    }

    void work() {
        System.out.println("工作");
    }
//这是对应的测试方法，当然题目可能是让我们对供人类，学生类，和班干部类进行测试。这个先当了解一下吧
    public static void main(String[] args) {
        //创建对象，用我们的构造函数。。
        Person person=new Person("牛马",99,"china","男");

        // 测试 getter 方法获取属性值
        System.out.println("Name: " + person.getName());
        System.out.println("Age: " + person.getAge());
        System.out.println("Country: " + person.getCountry());
        System.out.println("Gender: " + person.getGender());

        // 测试 setter 方法设置属性值
        person.setName("拉克丝");
        person.setAge(20);
        person.setCountry("LOL");
        person.setGender("女");

        // 再次使用 getter 方法获取更新后的属性值，验证我们的setter方法确实对属性进行了修改
        System.out.println("Updated Name: " + person.getName());
        System.out.println("Updated Age: " + person.getAge());
        System.out.println("Updated Country: " + person.getCountry());
        System.out.println("Updated Gender: " + person.getGender());
    }
}


```

## (2) 根据人类，派生一个**工人类**，增加属性：单位、工龄；重写工作方法（工人的工作是……）

```java
package hello.main;

// Worker 类继承自 Person 类，表示一个工人
public class Worker extends Person {
    // 新增属性：单位（公司）和工龄


    private String company;          // 工人所在的单位（公司）
    private int workAge;      // 工人的工龄

    // 构造方法，用于创建 Worker 对象，并初始化父类（Person）和子类的属性
    public Worker(String name, int age, String country, String gender, String company, int workExperience) {
        // 调用父类的构造方法，初始化继承自 Person 类的属性
        super(name, age, country, gender);

        // 初始化 Worker 类新增的属性
        this.company = company;
        this.workAge = workExperience;
    }

    // 新增属性的 getter 和 setter 方法，用于获取和设置工人的单位和工龄
    public String getCompany() {
        return company;
    }

    public void setCompany(String company) {
        this.company = company;
    }

    public int getWorkAge() {
        return workAge;
    }

    public void setWorkExperience(int workExperience) {
        this.workAge = workExperience;
    }

    // 重写工作方法，提供 Worker 类自己的实现,重写都是一个理。。。
    @Override
    void work() {
        System.out.println("工人在" + company + "工作，工龄：" + workAge + "年");
    }


    //这里是我们的测试方法。，。。
    public static void main(String[] args) {
        // 测试 Worker 类
        Worker worker = new Worker("张三", 30, "China", "男", "牛马制造厂", 5);

        // 测试 Worker 类新增的属性的 getter 方法
        System.out.println("工人所在的厂是 " + worker.getCompany());
        System.out.println("工人所在的厂是 " + worker.getWorkAge());

        // 测试 Worker 类的 work 方法（重写了父类的 work 方法）
        worker.work();
    }
}
```

## (3) 根据人类，派生一个**学生类**，增加属性：学校、学号；重写工作方法（学生的工作是学习）。

```java

// 学生类，继承自 Person 类
public class Student extends Person {
    private String school;  // 学生所在学校
    private String studentId;  // 学生学号

    // 构造方法，用于创建 Student 对象，并初始化父类（Person）和子类的属性
    public Student(String name, int age, String country, String gender, String school, String studentId) {
        // 调用父类的构造方法，初始化继承自 Person 类的属性
        super(name, age, country, gender);

        // 初始化 Student 类新增的属性
        this.school = school;
        this.studentId = studentId;
    }

    // 新增属性的 getter 和 setter 方法
    public String getSchool() {
        return school;
    }

    public void setSchool(String school) {
        this.school = school;
    }

    public String getStudentId() {
        return studentId;
    }

    public void setStudentId(String studentId) {
        this.studentId = studentId;
    }

    // 重写工作方法，提供 Student 类自己的实现
    @Override
    void work() {
        System.out.println("学生在" + school + "学习，学号：" + studentId);
    }

    //这里是测试方法。。。。。
    public static void main(String[] args) {
        // 测试 Student 类
        Student student = new Student("李四", 22, "China", "男", "zzuli", "123456");

        // 测试 getter 方法获取属性值
        System.out.println("Name: " + student.getName());
        System.out.println("Age: " + student.getAge());
        System.out.println("Country: " + student.getCountry());
        System.out.println("Gender: " + student.getGender());
        System.out.println("School: " + student.getSchool());
        System.out.println("Student ID: " + student.getStudentId());

        // 测试 setter 方法设置属性值
        student.setName("王五");
        student.setAge(21);
        student.setCountry("USA");
        student.setGender("女");
        student.setSchool("zzuliiii");
        student.setStudentId("789012");

        // 再次使用 getter 方法获取更新后的属性值，验证 setter 方法确实对属性进行了修改
        System.out.println("更新后的name " + student.getName());
        System.out.println("更新后的Age: " + student.getAge());
        System.out.println("更新后的Country: " + student.getCountry());
        System.out.println("更新后的Gender: " + student.getGender());
        System.out.println("更新后的School: " + student.getSchool());
        System.out.println("更新后的Student ID: " + student.getStudentId());

        
        // 测试重写的 work 方法
        student.work();
    }
}
```

```java
package hello.main;
```
